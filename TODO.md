# Flow
    Core
        Go to homepage (still keep it) -> go to dash board
    Dashboard
        choose or create new project -> setting for the project: the genre, api key, model, other confi, all config in -> create new novel -> click -> upload text to this project -> split chunk and show the chunk text as a table list, about 50 chunk per list(can be change in setting) -> can choose translate all or translate chunk you want -> show the time count, and progress bar-> show original chunk and translated chunk -> download the whole translated trunk or download the trunk you want
        Go to project you want -> see and change the glossary
        Go to project you want -> See translation progress bar for each novel in project 
        Show the log of the translation (can hide and show)
